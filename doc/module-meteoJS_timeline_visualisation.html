Class: module:meteoJS/timeline/visualisation

(abstract) module:meteoJS/timeline/visualisation(options)

Object to visualise module:meteoJS/timeline~Timeline. How to create your own visualisation object: * Inherit this object * Inherit method onChangeTime() * Inherit method onChangeTimes() * Inherit method emptyNode() * Inherit method onInitNode() * To get all times to display use getTimelineTimes() * Use method timeToText() to convert a Date to String. * Use method attachEventListener() to attach event listeners on a object. The event listener will be deleted automatically if the output is muted. * At the end of the constructor call "this.setNode(this.options.node);".
